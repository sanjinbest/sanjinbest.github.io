JDK内Function的使用举例

JDK8的Map接口提供了一个新方法computeIfAbsent，该方法接收2个参数key和Function。方法会判断传入的key所对应的Value是否在map内，如果Value不存在或为null，则调用Function接口生成一个Value并与key关联存入map中。
