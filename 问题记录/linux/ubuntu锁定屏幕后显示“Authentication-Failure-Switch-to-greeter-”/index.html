ubuntu锁定屏幕后显示“Authentication Failure Switch to greeter...”

发表于 2018-09-11 | 分类于 问题记录 ， linux | 阅读次数:
  • Ubuntu版本：Ubuntu 16.04.5 LTS
  • 问题描述：锁屏后登录框提示“Authentication Failure Switch to greeter…”
  • bug记录地址：Authentication Failure
  • 解决方法：在命令行执行如下命令：
    $ sudo apt install --reinstall lightdm
$ sudo service lightdm restart
    注意，第二个命令会停止当前打开的app和进程，请保存数据后在执行。
